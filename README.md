## 🛠️ Technology Stack & Architecture
* [cite_start]**Data Generation:** Python scripts simulating realistic, high-velocity e-commerce JSON payloads[cite: 29].
* [cite_start]**Ingestion (Apache Kafka):** Deployed as a highly scalable message broker to handle real-time streaming ingestion without data loss[cite: 547].
* [cite_start]**Stream Processing:** Engineered a pipeline to consume Kafka streams, flatten nested JSON payloads, and clean the data[cite: 548].
* [cite_start]**Data Warehouse (PostgreSQL):** Configured as a structured Data Warehouse to store highly organized, AI-ready tables[cite: 549].
* [cite_start]**AI Integration:** Integrated Docker Model Runner to query the data warehouse and prompt a local open-source Large Language Model (LLM) to generate personalized product recommendations[cite: 550].
