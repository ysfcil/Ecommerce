# Real-Time E-Commerce Recommendation Engine

## 📌 Project Overview
This project is a full-stack, real-time data architecture that spans from synthetic data generation all the way to a Generative AI integration]

I designed a zero-cost, local enterprise architecture utilizing Docker containers to ensure total environment isolation

## 🛠️ Technology Stack & Architecture
**Data Generation:** Python scripts simulating realistic, high-velocity e-commerce JSON payloads

**Ingestion (Apache Kafka):** Deployed as a highly scalable message broker to handle real-time streaming ingestion without data loss

**Stream Processing:** Engineered a pipeline to consume Kafka streams, flatten nested JSON payloads, and clean the data

**Data Warehouse (PostgreSQL):** Configured as a structured Data Warehouse to store highly organized, AI-ready tables
